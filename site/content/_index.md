# Python One

Historical Python source documentation, from 1.0.1 (February 1994) through
2.0c1 (October 2000). Each version's docs are converted from the original
LaTeX sources in the tarball and presented here as browsable Markdown.

The source tarballs come from
[legacy.python.org/download/releases/src](https://legacy.python.org/download/releases/src/).

Browse by version in the left sidebar, or jump straight to a release:
